== # How to use

Follow below steps and use it:

1. Add the "Brand" with affiliate brand name and affiliate program URL
2. Add the "Link" with your acutal affiliate link, select the brand and set the clocked link
3. Use the shortcode `[aff_link id='{ID}']Content[/aff_link]` to display the affiliate link
4. Also, Add the "Asset" with your acutal affiliate link, select the brand and set the clocked link
5. Use the shortcode `[aff_asset id='{ID}']` to display the affiliate assets.
